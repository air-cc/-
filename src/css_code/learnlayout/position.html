static

relative - 相对位置

相对于其本身的位置做调整, 但在页面中占据的空间仍为其原本的位置
这是其他元素

fixed - 固定位置

相对于浏览器的视窗来定位，不随页面滚动而调整位置，脱离文档流?
这是其他元素

absoulte - 绝对位置

相对于位置属性为非static的最近的父元素来定位，如果其父元素中没有非static的元素则相对的 body 元素来定位
